Set up:
1. Ensure all the necessary software is installed
    - Node.js v14 or above
    - GitHub Desktop/Git Bash 
    - Sqlite3
2. Clone repository from GitHub with the command:
    git clone git@github.com:unsw-cse-capstone-project/capstone-project-comp3900-w17a-kai-will-make-tony-rich.git

3. Navigate to the backend directory. Enter the command 'npm install'. 

4. Enter the command 'node server.js'. 

5. Open another terminal and then navigate to the frontend directory. Enter the command 'npm install'. 

6. Enter the command 'npm start'. 

Note: This should open the website automatically in the browser. 
To empty and restart the backend database, run the command 'npm run reset' in the backend directory.
